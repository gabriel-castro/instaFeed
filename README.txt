Get instagram feed from public accounts

HOW TO INSTALL:
- download the instaFeed.js file and put in ur website folder.
- In your HTML file put <script src="instaFeed.js"></script> in the head tag.

HOW TO USE:
- call getInstaFeed('public_instagram_account_here'); in your script

change the following vars:
myWebsite -> as string give ur website url (without 'https://')
numberOfFotos -> int, number of fotos the script will show from the account
